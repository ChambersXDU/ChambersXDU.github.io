hexo里的几个常用命令

  • npm install hexo -g #安装Hexo
  • npm update hexo -g #升级
  • hexo init #初始化博客
  • hexo n “我的博客” == hexo new “我的博客” #新建文章
  • hexo g == hexo generate #生成
  • hexo s == hexo server #启动本地预览
  • hexo d == hexo deploy #部署到远程
  • hexo clean #清除缓存，若网页正常,可以忽略
